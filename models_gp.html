My Experience:

Through out the making of this project Ive come to learn a lot of things reguarding web development. In order to create an attractive and working website one must first learn how to organise the elemnts with in the website. To manage the contents one must fully utilize the "div" function. It helps you manage the contents the way you like it. Giving adequete amount padding and margin is also very important. Looking at tutorials online and examples also helped me alot while making these webpages. Adding too many features to the same page makes the page complicated to manage. Keeping everything organised and simple is also important. The color combination should also be selected carefully as the user may not feel comfortable while using the web pages. Making these web pages helped me understand links between multiple webpages and how they worked which helped me understand the internet even more. There were multiple web pages that i viewd and many tutorials that i learned to make my web pages look better.
